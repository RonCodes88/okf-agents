# Account Setup

## Creating an Account

1. Go to acme.com/signup
2. Enter your email and choose a password
3. Verify your email address
4. Choose a plan (see [Billing](billing.md) for plan details)

## Initial Configuration

After signup, complete these steps:

- **Add a payment method** — required for paid plans (see [Billing](billing.md))
- **Invite team members** — Settings → Team → Invite
- **Set up notifications** — Settings → Notifications

## Common Setup Issues

If you run into problems during setup, see [Troubleshooting](troubleshooting.md).
